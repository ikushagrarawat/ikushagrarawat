## About Me

Backend-focused software engineer currently working at **Tata Consultancy Services (TCS)**, where I build systems around document validation, workflow automation, and structured-data-driven analysis. I enjoy building software that is not only functional, but also clear, maintainable, and useful in real business workflows.

My experience spans backend-heavy implementation in **Java + Spring Boot** and frontend/product work using **React.js, Next.js, and Tailwind CSS**. I like working at the intersection of engineering depth and product usability.
